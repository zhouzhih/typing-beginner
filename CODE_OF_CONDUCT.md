# Code of Conduct

## Our Pledge

我们希望这个项目保持友善、尊重和建设性。参与者应以帮助学习者和改进项目为目标。

## Expected Behavior

- 尊重不同经验水平的贡献者。
- 给出具体、可操作的反馈。
- 讨论问题本身，避免人身攻击。
- 保护儿童学习场景相关的隐私和安全。

## Unacceptable Behavior

- 侮辱、骚扰、威胁或歧视性言论。
- 发布他人的私人信息。
- 故意破坏项目或误导用户。
- 在问题或 PR 中发布恶意链接、恶意代码或不适合儿童的内容。

## Enforcement

维护者可以删除不当内容、关闭讨论、限制参与者继续互动。严重问题可通过 GitHub 的举报和安全渠道处理。
