# Contributing

欢迎给打字小课堂提交改进。这个项目面向小学生打字初学者，变更应优先保证简单、清晰、稳定。

## 开发流程

1. Fork 仓库并创建分支。
2. 安装依赖：

   ```bash
   npm ci
   ```

3. 本地启动网页版本：

   ```bash
   npm run dev
   ```

4. 提交前运行验证：

   ```bash
   make verify
   npm run lint
   ```

5. 如果改动涉及 Tauri、图标或打包，请额外运行：

   ```bash
   make mac-app
   ```

## 代码约定

- 保持界面文案简短，适合小学生和家长理解。
- 新关卡应循序渐进，避免突然增加太多陌生按键。
- 练习记录只保存在本机，不上传个人数据。
- 对修复 bug 或改变行为的改动，补充对应测试。

## Pull Request

PR 请说明：

- 改了什么。
- 为什么需要这个改动。
- 运行过哪些验证命令。

感谢参与。
